# 推薦系統的模型驗證方法

## 線下
### 排序指標
#### MRR@k(Mean Reciprocal Rank)
- 評估用戶的排序之中，第一個出現的正樣本位置的指標。
- 主要是加總所有用戶的一個出現正樣本位置的位置之倒數。
- 在排序指標中著重評估第一個正樣本，後續的排序不影響指標。

#### MAP@k(Mean Average Precision)
- 評估平均用戶的排序在前 Top k 個內，每一個正樣本位置 Precision@k 的平均值。
- 在排序指標中著重評估多個正樣本排序是否相對靠前，但當只有一個正樣本且在最相對靠前時，可能相對樂觀。

#### NDCG(Normalized Dicounted Cumulative Gain)
- 評估排序隨著位置權重遞減，且對比其他指標都是衡量二元結果(互動=1, 不互動=0)，可以考慮不同互動的權重，比如收藏=2, 喜歡=1, 不喜歡=0等等。
- 是非常主流的線性指標。

### 其他評估指標

#### 新奇性(Novelty)
- 評估用戶推薦清單對於該用戶是否新奇(沒看過、少看過)，通常有助於探索用戶偏好以及長期效益。
